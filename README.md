fedreporter Package:
====================

The goal of `fedreporter` is to provide downloads data from NIH 'ExPORTER'

Installation
------------

You can install `fedreporter` from GitHub with:

``` r
# install.packages("remotes")
remotes::install_github("muschellij2/fedreporter")
```
